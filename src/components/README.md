# Components Structure

## Canonical Buckets

- `ui/`: Shared primitive UI components.
- `layout/`: Navigation and shell components.
- `sections/`: Composed storefront section blocks.
- `store/`: Reusable store/product components.
- `admin/`: Reusable admin-only components.

## Rule of Thumb

- Add new reusable UI in canonical buckets.
- Keep each component in the bucket that matches its responsibility.
